# Job Board Interface

Simple React + Vite job board interface demo.

Requirements
- Node.js (16+)

Install and run

```bash
npm install
npm run dev
```

Features
- Display job cards
- Search by title (real-time)
- Filter by location and type
- Optional alphabetical sort
- Responsive layout

Folder structure

- src/components: `JobCard.jsx`, `Filters.jsx`
- src/data: `jobs.js`
- src/App.jsx: main app
